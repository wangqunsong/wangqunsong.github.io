概述

Socket是介于应用层与网络层之间的抽象层接口，其基于TCP/IP协议族并将复杂的TCP/IP协议族隐藏封装；我们只需要遵循socket标准去编程，就能编写出符合TCP/UDP的程序。或者也可将socket称为host+port。

####套接字

套接字最初主要用于进程间通信(IPC)，主要有两种类型的套接字：基于文件和面向网络。

  • 基于文件的套接字家族：AF_UNIX(后也称AF_LOCAL，但Python仍用AF_UNIX)

  • 面向网络的套接字家族：AF_INET

    Python 只支持 AF_UNIX、AF_NETLINK、AF_TIPC 和 AF_INET 家族

面向连接与无连接

正如前所述，socket是建立在TCP/UDP之上的，故拥有TCP/UDP的特性：面向连接和无连接。

  • 面向连接：SOCK_STREAM，基于流套接字
  • 无连接：SOCK_DGRAM，基于数据报(datagram)

工作流程

套接字一般用于客户端/服务器结构中，流程为：

    1. 服务器先初始化Socket，然后绑定(bind)IP与端口，并启动监听(listen)，调用accept阻塞，等待客户端的连接；

    2. 客户端初始化Socket，然后连接服务器(conect)，连接成功后，客户端开始向服务器发送请求数据，服务器收到并处理请求，然后把回应数据发送给客户端，客户端读取数据，最后关闭连接，本次交互结束。

通用

​ 要创建套接字，必须使用 socket.socket()函数，它一般的语法如下：

socket(socket_family, socket_type, protocol=0)
# 其中，socket_family是 AF_UNIX或 AF_INET，socket_type 是 SOCK_STREAM或 SOCK_DGRAM。protocol 通常省略，默认为0。

所以，为了创建 TCP/IP 套接字，可以用下面的方式调用 socket.socket()：

tcpSock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

同样，为了创建 UDP/IP 套接字，需要执行以下语句:

udpSock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

Socket函数

服务器套接字函数
s.bind()   #绑定IP + 端口
s.listen()  #TCP监听
s.accept()  #被动接受TCP客户的连接,(阻塞式)等待连接的到来
客户端套接字函数
s.connect()  #主动初始化TCP服务器连接
s.conect_ex() #connect()函数的扩展版本,出错时返回出错码,而不是抛出异常

#####普通套接字函数

s.recv()            #接收TCP数据
s.send()            #发送TCP数据(send在待发送数据量大于己端缓存区剩余空间时,数据丢失,不会发完)
s.sendall()         #发送完整的TCP数据(本质就是循环调用send,sendall在待发送数据量大于己端缓存区剩余空间时,数据不丢失,循环调用send直到发完)
s.recvfrom()        #接收UDP数据
s.sendto()          #发送UDP数据
s.getpeername()     #连接到当前套接字的远端的地址
s.getsockname()     #当前套接字的地址
s.getsockopt()      #返回指定套接字的参数
s.setsockopt()      #设置指定套接字的参数
s.close()           #关闭套接字
面向阻塞套接字函数
s.setblocking()     #设置套接字的阻塞与非阻塞模式
s.settimeout()      #设置阻塞套接字操作的超时时间
s.gettimeout()      #得到阻塞套接字操作的超时时间

#####面向文件套接字函数

s.fileno()          #套接字的文件描述符
s.makefile()        #创建一个与该套接字相关的文件

基于TCP套接字

#######服务端########
# -*- coding: utf-8 -*-
import socket
HOST = ''
PORT = 21123
BUFSIZE = 1024
ADDR = (HOST,PORT)

sock_tcp_server = socket.socket(socket.AF_INET,sock.SOCK_STREAM)
sock_tcp_server.bind(ADDR)
sock_tcp_server.listen(5)
while True:
    print("等待客户端的链接")
    conn,addr = sock_tcp_server.accept()
    print("接收到信息来自于{}".format(addr))
    while True:
        data = sock_tcp_server.recv(BUFSIZE)
        if not data:break
        sock_tcp_server.send(data.upper())
        sock_tcp_server.close()
sock_tcp_server.close()

#######客户端########
# -*- coding: utf-8 -*-
import socket
HOST = 'localhost'
PORT = 21123
BUFSIZE = 1024
ADDR = (HOST,PORT)

sock_tcp_client = socket.socket(socket.AF_INET,sock.SOCK_STREAM)
sock_tcp_client.connect_ex(ADDR)

while True:
    data = input('> ').strip()
    if not data: break
    sock_tcp_client.send(data.encode('utf-8'))
    accept_data = sock_tcp_client.recv(BUFSIZE)
    if not accept_data: break
    sock_tcp_client.close()

基于UDP套接字

#######服务端########
import socket

HOST = ''
PORT = 21567
BUFSIZ = 1024
ADDR = (HOST,PORT)

sock_udp_server = socket.socket(socket.AF_INET,socket.SOCK_DGRAM)
sock_udp_server.bind(ADDR)

while True:
    print("等待客户端的UDP链接")
    data,addr = sock_udp_server.recvfrom(BUFSIZ)
    print("收到的信息为{0}，类型为{1}".format(data,type(data)))
    sock_udp_server.sendto(data.upper(),addr)
sock_udp_server.close()

#######客户端########
# -*- coding: utf-8 -*-
import socket

HOST = 'localhost'
PORT = 21567
BUFSIZ = 1024
ADDR = (HOST,PORT)

sock_udp_client = socket.socket(socket.AF_INET,socket.SOCK_DGRAM)

while True:
    data = input('> ').strip()
    if not data:
        break
    sock_udp_client.sendto(data.encode('utf-8'),ADDR)
    accept_data,addr= sock_udp_client.recvfrom(BUFSIZ)
    if not accept_data:
        break
    print(accept_data.decode('utf-8'),addr)
sock_udp_client.close()

SocketServer模块

SocketServer是标准库中的一个高级模块（Python 3.x 中重命名为 socketserver）。

#######服务端########
# -*- coding: utf-8 -*-
from socketserver import TCPServer,StreamRequestHandler
from time import ctime
HOST = ''
PORT = 21567
ADDR = (HOST,PORT)

class MyRequestHandler(StreamRequestHandler):
    
    def handle(self):
        print("等待来自客户端{}的连接".format(self.client_address))
        for line in self.rfile:
            self.rfile.write(ctime(),line)
if __name__ == '__main__':
    tcp_server = TCPServer(ADDR,MyRequestHandler)
    tcp_server.serve_forever()

#######客户端########
# -*- coding: utf-8 -*-
from socket import *
HOST = '127.0.0.1'
PORT = 21567
BUFSIZE = 1024
ADDR = (HOST,PORT)

while True:
    tcp_client = socket(AF_INET,SOCK_STREAM)
    tcp_client.connect_ex(ADDR)
    data = b'Hello World!'
    if not data:
        break
    tcp_client.send(data)
    response_data = tcp_client.recv(BUFSIZE)
    if not response_data:
        break
    print("返回的信息为".format(response_data.strip()))
    tcp_client.close()
